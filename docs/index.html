Investigative Desk

An agent assembly line that turns messy public records into sourced, reviewable investigative leads — on any dataset. Every number traces back to a source; the whole thing re-runs offline.

What it is

Six composable Agent Skills organized into three functional buckets, plus a verifiability contract underneath. Scope the question with the user, then:

Scope (S0)
clarify the question → open a case file → pick the beat-pack
Gather + analyze
ingest + web-search for data · profile · discover · verify
Organize
case file: threads, status, cold-ledger, journal
Harden claims
check the law · footnote to primary sources · QA

Built for the lobbying corpus, but corpus-agnostic: S0 selects a beat-pack and the tool works on other public-records beats.

Install & run

Runs inside a code-executing agent (Claude Code, Cursor, or Cowork). Python standard library only; no keys needed for the core findings.

export GAIN_DATA_DIR=/path/to/data GAIN_WORKDIR=$PWD/build/workdir
python3 skills/lobbying-influence-mapper/scripts/ingest.py --years 2025 --datasets senate contributions press
python3 skills/lobbying-influence-mapper/scripts/resolve_entities.py
python3 skills/lobbying-influence-mapper/scripts/xref.py gatekeeper --filer lobbyist --year 2025 --top 12
python3 skills/investigative-method/tools/find_data.py "your topic"   # web-search for data
python3 setup_keys.py    # optional: add free API keys (Congress.gov, FEC, ...)
