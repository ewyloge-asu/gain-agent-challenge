Investigative Desk

A toolkit that lets an AI coding agent run a records investigation the way a newsroom would: gather the documents, find the leads, keep the investigation organized across sessions, and tie every claim to a source record a human can check.

We built it for the challenge's federal lobbying corpus and used it to find real stories — who lobbyist money actually pools around, a foreign-influence disclosure seam, and $80 million of fictional claims sitting unchecked in the official disclosure data — then packaged the whole workflow as Agent Skills any other team can install and point at their own data.

01 · The tool

What it is

Six Agent Skills that work as one pipeline. In plain terms, the agent walks the same four steps a careful reporter would:

1 · Scope the question
The agent interviews you: what's the question, which entities, what time frame, what counts as a finding? That scope becomes the case file's brief and picks the right tools for the beat.
2 · Gather & analyze
Deterministic scripts load the records into a database, resolve messy names to real entities, profile what's normal, and surface what isn't — including searching the web for outside datasets and APIs to check claims against.
3 · Stay organized
A case file on disk tracks every thread (open, chasing, confirmed, or cold), every key entity, and a session journal — so next week's session picks up exactly where this one stopped.
4 · Make it publishable
Before a claim ships: read the actual statute and apply it element-by-element, footnote the draft to primary sources as tracked changes, and flag anything a human still needs to judge.

Underneath all four steps, one rule: every number carries a provenance ID that traces back to the raw filing it came from, so an editor can audit any claim in one click.

02 · The journalism

What we found

The headline: lobbyist money pools around the members who control whether legislation moves. In 2025, the top recipient of contributions reported by registered lobbyists was Rep. Brett Guthrie — $694,295 from 177 different lobbying firms — and he chairs House Energy & Commerce. The rest of the leaderboard reads the same way: Ways & Means chair Jason Smith ($674,775), Appropriations chair Tom Cole, Financial Services chair French Hill, plus the floor leaders of both parties. Checked against each member's FEC totals, lobbyist money runs 13–18% of the entire campaign haul for the money-committee chairs. We claim the pattern, not causation — and we verified every recipient against the official congressional roster.

The same pipeline also surfaced a foreign-influence seam, an abuse of the disclosure system itself, and one lead we killed on purpose:

FindingStatus
Pay-the-gavel — lobbyist contributions concentrate on committee chairs and floor leaders (above)Confirmed
Foreign revolving door — Tencent's lobbying spend jumped 5× ($800K → $4.04M) while fighting a Pentagon designation, and the ex-White House official lobbying for it isn't in the foreign-agents registry — legal, but a seam a reporter would chaseConfirmed
Loc Nation — one Delaware filer claimed $80M of fictional "lobbying income" (four $20M quarters of pseudo-legal reparations claims), making it the #1 "client" in the entire 2025 data — nobody validates these filings; flagged to the panel as possible misuse of a federal systemIntegrity flag
Coordinated messaging — the tool detects when many offices push identical talking points in a tight window (e.g., a 9-member bipartisan cluster reusing one think-tank statistic within 2 days)Method
Senate↔House consistency — do firms tell the two chambers different numbers? We checked 36,643 matched filings: zero gaps over $100K. A clean dead end, documented so nobody re-chases itCold

How to read the labels: Confirmed = verified against outside records, every number sourced · Integrity flag = referred to the panel as a possible legal issue, not asserted as one · Method = a reusable detection capability more than a single story · Cold = a dead end we verified and recorded on purpose, so the investigation doesn't repeat it.

03 · The skills

The tools (six Agent Skills)

Each skill is useful on its own — install just one, or the whole line. An Agent Skill is a folder of instructions and scripts that a coding agent (Claude Code, Cursor, Cowork) picks up automatically.

The technical names below are the skills' install identifiers; the label on each card is what it does. One deserves a footnote: robodoig honors Steve Doig, the Pulitzer-winning pioneer of data journalism at ASU's Cronkite School — profiling the data before trusting it is his method, automated.

04 · Two minutes to first result

Try it

Requirements: a code-executing agent (Claude Code, Cursor, or Cowork) — or just a shell — and Python 3. No packages to install, no API keys needed for the core findings. Point GAIN_DATA_DIR at the challenge corpus (evaluators already have it; each skill also ships a small demo slice that works out of the box).

# from the unzipped menu bundle
export GAIN_DATA_DIR=/path/to/gain-corpus GAIN_WORKDIR=$PWD/workdir
python3 skills/lobbying-influence-mapper/scripts/ingest.py --years 2025 --datasets senate contributions press
python3 skills/lobbying-influence-mapper/scripts/resolve_entities.py
python3 skills/lobbying-influence-mapper/scripts/xref.py gatekeeper --filer lobbyist --year 2025 --top 12
#   ^ that's the pay-the-gavel leaderboard — Guthrie $694,295 — in about 30 seconds
python3 skills/lobbying-influence-mapper/scripts/xref.py anomaly --year 2025 --factor 5
#   ^ and that's Loc Nation's $80M flagged as the #1 outlier
python3 skills/investigative-method/tools/find_data.py "your topic"   # find outside datasets/APIs
python3 setup_keys.py    # optional: free API keys unlock live FEC / Congress.gov refreshes

06 · Generality

Not just this corpus

The lobbying work is one configuration, not the tool. To prove it, we ran the whole pipeline on a completely different question — "Which hospitals show billing patterns consistent with Medicare up-coding?" — using zero lobbying-specific code:

1 · Scoped the new beat
The scoping step took the healthcare question and generated a healthcare investigation plan — instead of falling back to lobbying.
2 · Found & kept the data
The data-search tool ranked CMS first, queried a live open-data portal, and surfaced the HHS Inspector General's up-coding report — then snapshotted the report and three live CMS data slices with checksums.
3 · Profiled & found leads
4,079 hospital billing rows profiled; per-provider highest-severity billing shares computed against the national baseline (median 74.4%) — top outliers bill 90–92%, exactly the pattern the OIG flags for up-coding review.
4 · Caught its own trap
The naive pass showed a wall of 100% shares — recognized as a data-suppression artifact, not a story; 120 providers excluded, method recorded. Ranked leads landed in a case file with next actions.

DimensionWhere it's demonstrated
